ABOUT PROJECT

In user interface (UI), it displays the name and head portrait of each student followed by the numbers of correct times, incorrect times, and attendances. Moreover, there are several buttons to realize different functions:
1. Randomly select student: this button aims to select student randomly, and the UI for the function will be individually designed. More exactly, there are three options for the selected student, the "Correct", "Incorrect", and "Dismiss".
2.Add student: sometimes, the professors may add new student into the list of class. In this case, the app sets this button for adding student. The new student will contain the data about first name, second name, and the gender.
3. Save files: After changing the original data, this function will be operated if the users intend to update the data. Meanwhile, all data will be saving into the CSV files.
4. Sorting options: there are five options for sorting the list, "first name", "second name", "absences", "percent correct", and "call frequency".
The whole data should be imported as CSV files, and the exported file type should be CSV as well.
